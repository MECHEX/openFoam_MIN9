# POD and EPOD Figures

This folder contains visual summaries generated by:

```powershell
python .\toolkit_test\plot_pod_epod.py
```

## POD Figures

- `pod_modal_energy.png` - modal energy and cumulative energy for velocity and heat flux.
- `pod_velocity_mean_field.png` - temporal mean velocity field.
- `pod_heat_flux_mean_field.png` - temporal mean wall heat-flux field.
- `pod_velocity_spatial_modes.png` - first four velocity POD spatial modes.
- `pod_heat_flux_spatial_modes.png` - first four heat-flux POD spatial modes.
- `pod_velocity_temporal_coefficients.png` - velocity POD modal amplitudes over snapshots.
- `pod_heat_flux_temporal_coefficients.png` - heat-flux POD modal amplitudes over snapshots.

## EPOD Figures

- `epod_reconstruction_quality.png` - target-field energy captured as more source modes are used.
- `epod_velocity_to_heat_flux_extended_modes.png` - heat-flux patterns associated with velocity POD timing.
- `epod_heat_flux_to_velocity_extended_modes.png` - velocity patterns associated with heat-flux POD timing.
- `epod_velocity_to_heat_flux_snapshot_reconstruction_mode3.png` - heat-flux target vs EPOD reconstruction using three velocity modes.
- `epod_heat_flux_to_velocity_snapshot_reconstruction_mode3.png` - velocity target vs EPOD reconstruction using three heat-flux modes.
