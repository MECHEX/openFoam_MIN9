# Archiwum

This directory stores older exploratory material that is no longer part of the active
working layout of the repository, but should remain available for traceability.

Typical contents:

- legacy pre-`VV_cases` studies
- exploratory mesh-generation assets
- pre-cleanup snapshots kept before restructuring an active study

Rules:

- archive, do not edit in place unless there is a specific recovery task
- active workflows should not write new results here
- if a file was moved here during cleanup, note that move in `VV_cases/RESEARCH_LOG.md`
