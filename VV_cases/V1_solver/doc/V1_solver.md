# V1 Solver Verification

## Objective

V1 is the hydrodynamic verification study for laminar flow past a confined circular cylinder.
The purpose is to verify that the OpenFOAM setup reproduces the onset of periodic vortex
shedding and the Strouhal number reported for confined channels in the literature.

The study contains two complementary layers:

- a beta = 0.50 mesh and streamwise-domain study used to select a numerically efficient
  reference setup
- a Poiseuille-inlet parity study against Sahin and Owens (2004) for selected blockage ratios

## Reference literature

- Sahin, M. and Owens, R.G. (2004), *A numerical investigation of wall effects up to high blockage ratios on two-dimensional flow past a confined circular cylinder*, Physics of Fluids 16, 1305-1320, doi:10.1063/1.1668285
- Lu, W., Chan, L. and Ooi, A. (2025), *Fluids* 10(4), 84, doi:10.3390/fluids10040084

## Geometry

### Cylinder and base dimensions

- Cylinder diameter: `D = 12 mm`
- Spanwise thickness for the 2D setup: `Lz = 10 mm`
- Cylinder center: `(x, y) = (0, 0)`

### Confined-channel verification family

For the Sahin and Owens parity cases:

- blockage ratio definition: `beta = D / H`
- upstream length: `8D`
- downstream length: `20D`
- `H = D / beta`

Cases already analysed in the repository include:

| beta | Channel height H [mm] | Role |
|---|---:|---|
| 0.300 | 40.0 | direct literature comparison |
| 0.375 | 32.0 | geometry-relevant interpolation case |
| 0.500 | 24.0 | direct literature comparison |
| 0.600 | 20.0 | additional confinement check |

### Streamwise-domain sensitivity case

For the beta = 0.50 streamwise-domain check:

- baseline domain: `8D` upstream, `20D` downstream
- extended domain: `10D` upstream, `30D` downstream

Only the streamwise extent was changed; channel height remained fixed because it defines
the physics through `beta`.

## Boundary conditions

### Velocity

- inlet: parabolic profile
- outlet: `zeroGradient`
- top wall: `noSlip`
- bottom wall: `noSlip`
- cylinder: `noSlip`
- front/back: `empty`

For the verification runs the Reynolds number convention is:

- `Re = U_max D / nu`
- `U_bulk = (2/3) U_max`

### Pressure

- outlet: fixed reference pressure
- inlet and walls: `zeroGradient`
- front/back: `empty`

## Numerical setup

- solver: `pimpleFoam`
- flow model: laminar, incompressible, Newtonian
- density for coefficient normalization: `rho = 1.205 kg/m^3`
- kinematic viscosity: `nu = 1.516e-5 m^2/s`

### Time integration and discretization

- time scheme: `backward`
- momentum convection: `Gauss linearUpwind grad(U)`
- laplacian: `Gauss linear corrected`
- gradient: `Gauss linear`

### Pressure-velocity coupling

- algorithm: `PIMPLE`
- `nOuterCorrectors = 2`
- `nCorrectors = 2`
- `nNonOrthogonalCorrectors = 1`
- residual control:
  - `U <= 1e-5`
  - `p <= 1e-5`

### Monitoring

- force coefficients on the cylinder:
  - `Cd`
  - `Cl`
  - shedding frequency `f`
  - `St = f D / U_max`
- solver residuals through `solverInfo`
- Courant number through `CourantNo`

## Mesh and mesh-independence study

### Mesh strategy

The mesh is generated by `blockMesh` and `snappyHexMesh` with local refinement near the
cylinder and in the wake.

The key tested variants for beta = 0.50 at `Re = 160` were:

| case | mesh | upstream | downstream | cells | regime | St |
|---|---|---:|---:|---:|---|---:|
| baseline_coarse_Re160 | coarse | 8D | 20D | 8598 | periodic | 0.3318 |
| baseline_medium_Re160 | medium | 8D | 20D | 44492 | periodic | 0.3421 |
| long_medium_Re160 | medium | 10D | 30D | 46040 | periodic | 0.3422 |
| long_target100k_Re160 | target100k | 10D | 30D | 104104 | periodic | 0.3459 |

### Interpretation

- The coarse mesh is too diffusive and underpredicts `St`.
- The difference between `baseline_medium_Re160` and `long_medium_Re160` is negligible.
- Increasing the cell count from about `46k` to about `104k` changes `St` by about `1.1%`.

### Recommended reference mesh

The recommended practical reference setup for V1 is:

- medium mesh
- baseline streamwise length `8D / 20D` for routine sweeps
- extended streamwise length `10D / 30D` only as a control case

This gives a good compromise between cost and accuracy.

## Convergence and monitored quantities

The acceptance criteria used in V1 are:

- residual monitoring through `solverInfo`
- physically stable force histories over the analysed time window
- periodic `Cl(t)` before extracting `f` and `St`

For periodic cases the Strouhal number is extracted from the dominant frequency in the
late-time `Cl` signal.

## Comparison with literature

The V1 parity study is already close to the Sahin and Owens Strouhal data.
For the directly matched cases at `beta = 0.30` and `beta = 0.50`, the current
discrepancy in `St` is on the order of `0.1%` to `0.7%` once the flow is periodic.

The `beta = 0.375` cases are used as interpolated geometry-relevant points rather than
strict direct literature matches.

## Recommended reference setup

Two V1 reference usages are recommended:

1. For efficient internal checks:
   - `beta = 0.50`
   - medium mesh
   - baseline domain `8D / 20D`
2. For article-grade literature parity:
   - Poiseuille inlet
   - no-slip walls
   - medium mesh
   - direct comparison at `beta = 0.30` and `beta = 0.50`
